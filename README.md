# Lab-Create-and-Manage-Users-in-Microsoft-Entra-ID-Azure-Active-Directory-
This lab demonstrates how to create and manage users in Microsoft Entra ID (formerly Azure Active Directory).

## Objectives
- Create a user
- Reset a password
- Create a security group
- Add users to groups
- Assign a license
- Delete a user

 ## Technologies Used
- Microsoft Azure
- Microsoft Entra ID
- Azure Portal

## Skills Demonstrated
- Identity and Access Management (IAM)
- User Administration
- Group Management
- Password Administration
- Microsoft Entra ID
